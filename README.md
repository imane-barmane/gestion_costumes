Application de Gestion de Costumes

Application mobile Flutter avec backend Laravel (API REST) pour la gestion de costumes.

Fonctionnalités
Pour les Clients

Visualisation de tous les costumes disponibles

Filtre des costumes selon les prix

Interface utilisateur intuitive

Pour les Vendeurs

Authentification par email / mot de passe

Upload d’images de costumes

Ajout, modification et suppression de costumes

Gestion de la description et du prix
